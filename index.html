    Introduction.

    GoogleFaces is a JSF library which provides a set of JSF components to use Google Charts in your JSF application.

    Example.

        <gf:pieChart title="TestChart" data="#{dataProvider.testData}" 
              var="data" value="#{data.value}" label="#{data.label}" />
